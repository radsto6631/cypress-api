# Cypress API Reference Project

A small Cypress project focused on basic API usage, data preparation, and simple end-to-end flows.

This repository acts mainly as a personal reference for common Cypress patterns when working with APIs.

It contains examples of authentication, creating and deleting data through API calls, intercepting requests, and mixing API setup with UI interactions.
